Methodology

RCA (Root Cause Analysis): 5 Whys

RCA keeps asking “Why?” until we stop treating the symptom and uncover the deeper cause that must be fixed.

Observed Failure

User cannot login to the system

Root Cause

Missing finally block causes connections not to be released properly.

Corrective Action

Add a finally block and create an automated test for connection leaks.
