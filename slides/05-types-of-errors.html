Three Types of System Errors

  • Syntax Errors
    Mistakes that violate language rules, typically detected during compilation.
    Examples: Missing semicolons, unmatched brackets, misspelled keywords.
  • Runtime Errors
    The program compiles successfully but crashes at runtime.
    Examples: Division by zero, invalid memory access, out of memory.
  • Logic Errors
    Flaws in logic that lead to incorrect results without crashing. Hardest to detect and fix.
    Examples: Sorting algorithm reversed, missing edge cases.
